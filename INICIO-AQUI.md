# Bem-vindo ao seu Mastermind AIOX 🚀

Siga esse guia **passo a passo** antes de começar.
Não pule etapas — cada uma depende da anterior.

---

## PASSO 1 — Crie uma conta no GitHub

O GitHub é onde ficam salvos todos os seus arquivos do Mastermind.

1. Acesse: **github.com**
2. Clique em **Sign up**
3. Crie sua conta com seu email
4. Confirme o email que chegar na sua caixa de entrada

> Depois de criar sua conta, avise a Julia para te dar acesso ao seu repositório.

---

## PASSO 2 — Instale o GitHub Desktop

O GitHub Desktop é o programa que você vai usar para baixar e salvar seus arquivos.
Não precisa digitar nenhum comando — tudo é por clique.

1. Acesse: **desktop.github.com**
2. Clique em **Download for Windows** (ou Mac, se for o seu caso)
3. Abra o arquivo baixado e instale
4. Abra o GitHub Desktop e faça login com sua conta do GitHub

---

## PASSO 3 — Instale o Node.js

O Node.js é um programa que roda alguns scripts do Mastermind nos bastidores.
Você não vai interagir com ele — só precisa estar instalado.

1. Acesse: **nodejs.org**
2. Clique no botão verde escrito **LTS** (é a versão estável)
3. Abra o arquivo baixado e instale (pode clicar em "Next" em tudo)

---

## PASSO 4 — Instale o Claude Code

O Claude Code é o programa principal — é por ele que você vai conversar com os agentes do Mastermind.

1. Acesse: **claude.ai/download**
2. Baixe e instale para o seu sistema
3. Faça login com sua conta da Anthropic (ou crie uma em **claude.ai**)

---

## PASSO 5 — Baixe o seu repositório

Agora você vai baixar os arquivos do seu Mastermind para o seu computador.

1. Abra o **GitHub Desktop**
2. Clique em **File → Clone repository**
3. Vá na aba **URL**
4. Digite: `gringoles777/mastermind-SEU-NOME`
   _(substitua SEU-NOME pelo seu nome, ex: `mastermind-alexandre`)_
5. Escolha uma pasta no seu computador onde quer salvar
6. Clique em **Clone**

Pronto — os arquivos estão no seu computador!

---

## PASSO 6 — Abra no Claude Code

1. Abra o **Claude Code**
2. Clique em **Open folder** (ou "Abrir pasta")
3. Navegue até a pasta onde você salvou o repositório no Passo 5
4. Selecione a pasta e abra

Você vai ver os arquivos do Mastermind carregados. Está tudo pronto para começar!

---

## Como salvar seu trabalho

Tudo que você fizer fica salvo localmente no seu computador.
Para salvar também no GitHub (recomendado — é seu backup na nuvem):

1. Abra o **GitHub Desktop**
2. Você vai ver as mudanças que fez listadas
3. Escreva uma descrição rápida do que fez (ex: "Primeiro workspace")
4. Clique em **Commit to master**
5. Clique em **Push origin**

Feito! Seu trabalho está salvo e seguro.

---

## Como receber atualizações do Mastermind

Quando o conteúdo base for atualizado, para receber as novidades:

1. Abra o **GitHub Desktop**
2. Clique em **Fetch origin**
3. Se aparecer **Pull origin**, clique nele também

Pronto — seu repositório está atualizado.
